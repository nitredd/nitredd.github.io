Use Atlas Search with Spring Data

With Atlas Search, you can take advantage of an Apache Lucene based index, an inverted index, to perform text-search or an open-ended analytical query when hosting your database with MongoDB. Spring Data doesn't support the Atlas $search aggregation pipeline stage, but this add-on library fills the gap
